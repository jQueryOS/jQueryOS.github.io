What AvocadOS?

Avacados, stylized as AvocadOS, is a minimal operating system designed to run almost pure jQuery.

The name?

Formerly jQueryOS, the name was changed to AvocadOS. It is based on a meme from Programming Puzzles & Code Golf:

hello i have quetions how does juice an avocado i have try for thirtee minut and no juic

user46049, avocado juice fanatic

The AvocadOS Manifesto

AvocadOS is great and does all the things.

  1. jQuery is the best-quality code ever. If you don't use jQuery you're an idiot.
  2. Only jQuery, otherwise not enough jQuery!
  3. jQuery is definitely the way to go.
